Help


This is a web app for students!

When you use this service, please login from url.

You read extensive reading, and record in this web app. How to record is only push button 'add' ,and fill out the form(date, title, series, and so on). keep reading and recording every day. Continuity is the father of success!

~Description of each function~

calendar: The number of words in the book is displayed on the date you read it. You can edit and delete registered book on this page. Click the number of words to see the details of the registered book.

statistics: You can see and Tweet your performance in statistics. Be able to read books with many words gradually!

profile: You can see your profile information. Please go to this page to unsubscribe(= delete user). But don't do it!

help: This page.
